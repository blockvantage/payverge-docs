# Scanning QR Codes

Welcome! This guide shows you how to access restaurant menus and place orders using Payverge's QR code system. No app download required!

## What is a QR Code?

A QR code is a square barcode that you can scan with your phone's camera. At Payverge restaurants, each table has a unique QR code that gives you instant access to the menu.

## How to Scan

### Using iPhone (iOS)

1. **Open Camera App**
   - Use the built-in Camera app (not a third-party app)
   - No special settings needed

2. **Point at QR Code**
   - Hold your phone 6-12 inches from the code
   - Make sure the entire code is visible
   - Camera will automatically detect it

3. **Tap the Notification**
   - A notification banner appears at the top
   - Shows the restaurant name
   - Tap to open in Safari

4. **Start Browsing**
   - Menu opens in your browser
   - No app download needed!

### Using Android

1. **Open Camera App**
   - Use your default Camera app
   - Or Google Lens if available

2. **Point at QR Code**
   - Hold phone 6-12 inches away
   - Ensure good lighting
   - Keep code in frame

3. **Tap the Link**
   - Link appears on screen
   - May say "Open with Chrome" or "Open link"
   - Tap to open

4. **Start Browsing**
   - Menu loads in your browser
   - Ready to order!

## Troubleshooting

### QR Code Won't Scan

**Try These Solutions:**

1. **Improve Lighting**
   - Move to better light
   - Avoid shadows on code
   - Turn on flashlight if needed

2. **Adjust Distance**
   - Move closer or farther
   - Try 6-12 inches away
   - Keep phone steady

3. **Clean the Code**
   - Wipe any dirt or moisture
   - Ensure code isn't damaged
   - Ask staff for new code if needed

4. **Check Camera**
   - Make sure lens is clean
   - Close other apps
   - Restart camera app

---

### Camera Doesn't Detect Code

**Check Settings:**

**iPhone:**
1. Go to Settings > Camera
2. Enable "Scan QR Codes"
3. Try scanning again

**Android:**
1. Open Camera settings
2. Enable "Google Lens suggestions"
3. Or download Google Lens app

---

### Link Doesn't Open

**Solutions:**

1. **Tap the Notification**
   - Don't just scan - tap the link
   - Look for banner at top of screen

2. **Manual Entry**
   - Ask staff for table URL
   - Type in browser
   - Format: payverge.io/t/[code]

3. **Try Different Browser**
   - Use Safari (iPhone)
   - Use Chrome (Android)
   - Avoid in-app browsers

---

### Code Says "Table Not Available"

**Possible Reasons:**

- Table is inactive
- Restaurant is closed
- Technical issue
- Wrong QR code

**Solution:**
- Ask staff for assistance
- They can activate the table
- Or provide correct code

## Alternative Access Methods

### If You Can't Scan

**Option 1: Manual URL**
1. Ask staff for table URL
2. Type in browser: `payverge.io/t/[code]`
3. Replace `[code]` with your table code

**Option 2: NFC Tap (if available)**
1. Some tables have NFC tags
2. Tap phone to tag
3. Link opens automatically

**Option 3: Staff Assistance**
1. Ask server for help
2. They can create bill manually
3. You can still view and pay

## Frequently Asked Questions

### Do I need to download an app?
No! Everything works in your mobile browser. No app download required.

### Can multiple people scan the same code?
Yes! Everyone at the table can scan the same QR code. Each person gets their own view but orders go to the same bill.

### What if I accidentally close the browser?
Just scan the code again! Your session and orders are saved.

### Does this work on all phones?
Yes! Works on iPhone, Android, and any phone with a camera and browser.

### Do I need to create an account?
No account needed! Just scan and start ordering.

### Is my payment information secure?
Yes! We use blockchain technology and encrypted connections for maximum security.

### Can I use this for takeout?
Yes! Some restaurants have QR codes at the counter for takeout orders.

### What if the QR code is damaged?
Ask your server for a new code or the table URL to type manually.

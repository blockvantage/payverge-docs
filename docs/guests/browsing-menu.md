# Browsing the Menu

After scanning the QR code, you can browse the restaurant's digital menu. This guide shows you how to explore items, view details, and find what you're looking for.

## Menu Layout

The menu is organized into categories for easy browsing:

- **Appetizers** - Starters and small plates
- **Main Courses** - Entrees and main dishes
- **Sides** - Side dishes and accompaniments
- **Desserts** - Sweet treats
- **Beverages** - Drinks and cocktails
- **And more** - Categories vary by restaurant

Choose how many:

- **Minus Button (-)** - Decrease quantity

- **Plus Button (+)** - Increase quantity

## Troubleshooting

### Menu Won't Load

**Solutions:**
1. Check internet connection
2. Refresh the page
3. Clear browser cache
4. Try different browser
5. Ask server for assistance

### Photos Not Showing

**Solutions:**
1. Wait for loading
2. Check connection speed
3. Scroll past and back
4. Refresh page

### Can't Find Item

**Solutions:**
1. Use search function
2. Check all categories
3. Ask server
4. Item may be sold out

### Prices Look Wrong

**Check:**
- Currency setting
- Tax inclusion
- Service fees
- Exchange rate

## Frequently Asked Questions

### Can I see nutritional information?
Some restaurants provide nutritional details. Tap item for full information.

### Are the photos accurate?
Photos show actual dishes from the restaurant. Presentation may vary slightly.

### Can I order from multiple categories?
Yes! Order as many items as you like from any category.

### How do I know if something is spicy?

### Can I modify items?
Yes! Use options, add-ons, and special requests to customize.

### What if I have allergies?
Check dietary tags and add allergy notes in special requests. Always inform your server.

### Can I save favorites?
Currently not available, but you can easily find items using search.

This varies by restaurant. Check the bill for final total with tax.
